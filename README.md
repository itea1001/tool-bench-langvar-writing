# Tool-Use Benchmark Language Variation

This repository contains results and analysis for our language variation experiments on tool-use benchmarks.

## Project Overview

We investigate how LLM performance on tool-use/function-calling benchmarks varies across languages.

### Benchmarks
- ComplexFuncBench
- BFCL (Berkeley Function Calling Leaderboard)
- tau2-bench

### Languages
- High-resource: English (en), Spanish (es), French (fr), Chinese (zh)
- Low-resource: Vietnamese (vi), Swahili (sw)
